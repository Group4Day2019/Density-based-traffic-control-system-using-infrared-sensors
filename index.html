The Current System
The current System is a one way because it uses only one fuctionality which is the fixed time delay
How is this problem

Its a problem in such away that during the rush hours you can find a green light on an empty road yet a road flooded with Vechicles has a red light on it

Good News

Our system is a two in one why because it has two functionalities which are both the fixed time delays and infrared sensors which are used to detect density of vechicles on the road

Simple description of how the System works

if its normal traffic flow at the junction our system uses fixed time delays to control the movement vechicles on the road. If vechicles start increasing on some roads at the junctions then the sensors come into play
